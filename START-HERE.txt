   → Verify AMPPS MySQL is running
   → Check phpMyAdmin is accessible

   → Check server console for errors
   → Run: node verify-mysql-migration.js

❌ "Data not saving"
   → Check MySQL connection in db.js
   → Verify database 'ratings' exists
   → Check user privileges

MIGRATION DETAILS
============================================================

Database Changes:
  ✅ PostgreSQL → MySQL (AMPPS)
  ✅ Connection pool: pg → mysql2/promise
  ✅ Parameter style: $1, $2 → ?
  ✅ Result format: .rows[0] → [rows][0]

SQL Syntax Changes:
  ✅ String concat: || → CONCAT()
  ✅ Regex: ~ → REGEXP
  ✅ JSON: ->> → JSON_EXTRACT()
  ✅ Insert ID: RETURNING → insertId
  ✅ Conflicts: ON CONFLICT → INSERT IGNORE
