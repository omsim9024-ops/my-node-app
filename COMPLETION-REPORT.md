## 💡 Customization Guide

### Change Primary Color
1. Open `enrollment-form.css`
2. Find `#1a472a` and `#2d7a3a`
3. Replace with your color
4. Save and refresh

### Adjust Spacing
1. Find `margin-bottom: 45px` or `gap: 24px`
2. Change values proportionally
3. Test on mobile and desktop
4. Save and refresh

### Modify Typography
1. Find `font-family:` at top of CSS
2. Change to your preferred font
3. Adjust `font-size:` values if needed
4. Save and refresh

See **DESIGN-SYSTEM.md** for detailed instructions
