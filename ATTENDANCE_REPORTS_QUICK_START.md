# Attendance Reports - Quick Reference

## 📊 Generate Your First Report

### Step 1: Navigate to Reports
- Click **📈 Reports** in the sidebar
- You'll see the Reports section with Attendance Report tab active

### Step 2: Select Your Section
- Dropdown shows only **your assigned sections**
- If assigned to only one section, it auto-selects

### Step 3: Choose Date Range
**Option A - Quick Filters**
- Click **Today** - Current date
- Click **This Week** - Mon-Today
- Click **This Month** - 1st-Today
- Click **This Quarter** - Quarter start-Today

**Option B - Custom Dates**
- Use "From Date" picker
- Use "To Date" picker
- Choose exact dates

### Step 4: Pick View Type
- **By Date**: See attendance chronologically
- **By Student**: See students with attendance rates

### Step 5: Generate
- Click **🔍 Generate Report**
- Results appear below with:
  - Summary statistics
  - Detailed attendance table
  - Filters and sort options

## 🎯 Common Tasks

### Find Students with Low Attendance
1. Select your section
2. Set date range
3. Select "By Student" view
4. Generate report
5. Look for **orange-highlighted rows** (< 75% attendance)

### Export Data
1. Generate a report
2. Choose export option:
   - **📊 Export Excel** → Downloads CSV file
   - **📄 Export PDF** → Creates PDF document
   - **🖨️ Print** → Opens print window

### Search Specific Student
1. Generate a report
2. Use **search box**: Type name or LRN
3. Filter **status**: Check/uncheck Present/Absent/Late
4. Results update automatically

### Sort Results
1. Generate a report
2. Use **Sort by** dropdown:
   - Sort by Date
   - Sort by Name (A-Z)
   - Sort by Attendance (by-student only)

## 📈 Summary Statistics Explained

| Metric | Meaning |
|--------|---------|
| **School Days** | Days with attendance records in your date range |
| **Total Present** | Count of students marked as present |
| **Total Absent** | Count of students marked as absent |
| **Total Late** | Count of students marked as late |
| **Attendance Rate** | (Present + Late) ÷ Total × 100% |

## 🎨 Legend

### Status Indicators
- 🟢 **Green/✅ Present** - Student was present
- 🔴 **Red/❌ Absent** - Student was absent
- 🟡 **Yellow/⏰ Late** - Student was late

### Visual Warnings
- 🟠 **Orange Background** - Student has < 75% attendance
- ⚠️ **Warning Icon** - Attendance below 75%

## ❓ FAQs

**Q: Can other advisers see my reports?**  
A: No. You can only see reports for your assigned sections.

**Q: Can I edit attendance in Reports?**  
A: No. Reports are read-only. Edit attendance using "Take Attendance" feature.

**Q: How far back can I report?**  
A: You can report on any dates where attendance was recorded.

**Q: Are reports saved automatically?**  
A: They're generated on-demand. Use Export to save permanently.

**Q: Can I share exported reports?**  
A: Yes! Downloaded Excel/PDF files can be emailed or printed.

**Q: What if I don't see any data?**  
A: Check that attendance was recorded for that date/section using "Take Attendance".

| Problem | Solution |
