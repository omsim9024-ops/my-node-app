## 🎨 Key Design Decisions

### Color Scheme
- **Primary**: Deep green (#1a472a - #2d7a3a)
- **Rationale**: Professional, educational, calming
- **Benefits**: Better than orange, more modern

### Typography
- **Font Stack**: System fonts (-apple-system, BlinkMacSystemFont, etc.)
- **Rationale**: Better performance, native feel
- **Benefits**: Faster loading, consistent across platforms

### Spacing
- **Base Unit**: 24px for gaps, 45px for sections
- **Rationale**: 8px multiples for consistency
- **Benefits**: Better proportions, easier maintenance

### Border Radius
- **Standard**: 16px on main containers
- **Rationale**: Modern, not too extreme
- **Benefits**: Contemporary look while maintaining readability

### Shadows
- **Layered System**: Multiple shadow sizes
- **Rationale**: Depth perception, visual hierarchy
- **Benefits**: Professional appearance, clear layering
