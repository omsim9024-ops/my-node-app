# Adviser System - Quick Start Guide

## 🎯 Quick Navigation

### For Administrators
- **Create Advisers:** `/admin-adviser-management.html` → Create Adviser tab
- **Manage Advisers:** `/admin-adviser-management.html` → Advisers List tab
- **Assign Sections:** `/admin-adviser-management.html` → Section Assignments tab

### For Advisers
- **Login:** `/auth.html?role=adviser`
- **Dashboard:** `/adviser-dashboard.html` (after login)

---

## 📋 Admin Quick Tasks

### Create a New Adviser Account
1. Open `/admin-adviser-management.html`
2. Click "Create Adviser" tab
3. Fill in:
   - **Adviser ID:** e.g., `ADV001`
   - **First Name:** e.g., `John`
   - **Last Name:** e.g., `Doe`
   - **Email:** e.g., `john.doe@cnhs.edu`
   - **Password:** At least 8 characters (e.g., `SecurePass123!`)
   - **Phone:** Optional
4. Click "Create Adviser"
5. Account is ready to use immediately

### Assign Sections to Adviser
1. Click "Section Assignments" tab
2. Select adviser from dropdown
3. Select section to assign
4. Select school year
5. Click "Assign Section"
6. Adviser can now access that section in their dashboard

### Edit Adviser Details
1. Click "Advisers List" tab
2. Search for adviser by name or email
3. Click "Edit" button
4. Modify status (Active/Inactive)
5. Click "Save Changes"

### Deactivate Adviser Account
1. Click "Advisers List" tab
2. Find adviser
3. Click "Delete" button
4. Account will be marked as inactive

## 👨‍🎓 Adviser Quick Tasks

### Log In
1. Go to `/auth.html?role=adviser`
2. Enter email and password
3. Click "Sign In"

### View My Sections
1. Click "My Sections" in sidebar
2. See all assigned sections
3. Click "View Class" to see students in that section

### Record Attendance
1. Click "Attendance" in sidebar
2. Select section from dropdown
3. Select date
4. Click ✓ to mark as Present, or use dropdown for other statuses
5. Add remarks if needed
6. Click "Save Attendance"

### Add Advising Notes
1. Click "Advising Notes" in sidebar
2. Select student from dropdown
3. Select note type (Counseling, Behavioral, Academic, Personal)
4. Write your note
5. Check "Mark as Confidential" if needed
6. Click "Add Note"

### View Student Profile
1. Click "Students" in sidebar
2. Select section from dropdown (or search)
3. Find student in list
4. Click "View" button to see full profile

### Check Notifications
1. Click "Notifications" in sidebar
2. View all recent notifications
3. Click "Mark as Read" on notifications

### Change Password
1. Click "Profile & Settings" in sidebar
2. Find "Change Password" section
3. Enter current password, new password, and confirm
4. Click "Change Password"

---

## 🔐 Security Notes

- **Passwords:** Minimum 8 characters. Keep secure and don't share.
- **Session:** Logged-in session expires when browser closes.
- **Confidential Notes:** Marked as confidential by default - not visible to students.
- **Only Your Data:** Advisers can only see sections and students assigned to them.

## 🚨 Troubleshooting

| Problem | Solution |
|---------|----------|
| "Invalid email or password" | Check caps lock; verify email is correct |
| "No sections assigned" | Admin must assign sections first |
| "Section not found" | Refresh page and select again |
| "Attendance not saving" | Select both section and date; at least one student |
| "Notes disappeared" | Select the student again from dropdown |
