# Standard Reports - Quick Reference Guide

## 🎯 What Changed?

### Before
- PDF export buttons on every report
- Static, plain text reports
- No customization options
- Required external PDF library

### After
- Print buttons with native browser printing
- Professional formatted reports with headers/footers
- Fully customizable report appearance
- No external dependencies needed
- Save as PDF directly from print dialog

---

## 🖨️ How to Print a Report

### Quick Steps:
1. Go to **Reports & Analytics** section
2. Select the report type (Demographics, Disability, etc.)
3. Click **🖨️ Print** button
4. Customize appearance in print preview if needed
5. Click **Print** or **Save as PDF**

---

## ⚙️ How to Customize Reports

### Toggle Customization Panel:
```
Click: ⚙️ Customize Report
```

### Available Options:

**Header Section**:
- 📝 School Name
- 📍 School Address
- 📄 Report Title
- 📅 Include Date (checkbox)
- 🎓 Include School Logo (checkbox)

**Footer Section**:
- 📋 Footer Text
- 🔢 Show Page Numbers (checkbox)

### Save Changes:
```
Click: Apply Settings
```

### Reset to Default:
```
Click: Reset to Defaults
```

## 💾 Settings Storage

- Settings are saved during your **current session**
- Settings are cleared when you close/refresh the page
- To keep custom settings, note them down or use defaults

---

## 🔧 Troubleshooting

### Print window didn't open?
- Check browser pop-up blocker
- Allow pop-ups for this website

### PDF won't save from print?
- Use **Save as PDF** option in print dialog
- Some browsers call it "Print to PDF"

### Customization not working?
- Click **Apply Settings** after changes
- Check browser console for errors (F12)

### Want default settings back?
- Click **Reset to Defaults** button
- Or manually clear customization fields

## 💡 Pro Tips

1. **Better Print Quality**: Print in **Color** mode for professional appearance
2. **Save Template**: Customize once, settings persist for session
3. **Batch Reports**: Print multiple reports with same settings
4. **PDF Archive**: Save PDFs to your computer for records
5. **Print Preview**: Always check print preview before printing
6. **Margins**: Adjust margins in print settings if needed
7. **Page Orientation**: Landscape recommended for tables with many columns

## 🔐 Data Security

- Report data includes only **Approved students**
- Pending and rejected enrollments are excluded
- Reports are generated in memory, not stored
- Customization settings are session-based
